# Lindo Tools Website

Official website for **Lindo Tools** — Professional Cordless Power Tools by Sichuan Lingdu Trading Co., Ltd.

## 🌐 Live Website
- **Production**: [www.lindolithiumtools.com](https://www.lindolithiumtools.com)

## 📌 About
This repository contains the source files for the Lindo Tools website.  
The site is built with static HTML + Tailwind CSS and hosted on Cloudflare Pages.

## 🛠️ Tech Stack
- HTML5 + Tailwind CSS
- Fully responsive design
- Hosted on Cloudflare Pages

## 📁 Project Structure
- `index.html` — Homepage
- `products.html` — Product listing
- `product-detail-*.html` — Product detail pages (Professional, DIY, Garden)
- `about.html`, `contact.html`, `request-quote.html` — Support pages
- `battery-platform.html`, `oem.html`, `private-label.html` — B2B pages

## 🚀 Getting Started
1. Download or clone this repository
2. Open `index.html` in your browser to preview locally
